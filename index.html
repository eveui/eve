What Is EVE?

EVE is planned to be a visualization library that includes charts, diagrams, pivot tables and grids. Eve's charts and diagrams are built with d3.js framework. EVE will save your time by letting you skip learning and coding with D3 and yet it will allow you to customize everything whatever you need.

Chart and Diagram Features

  • Highly customizable and extensible
  • Powerful and robust
  • Active development community
  • 21 supported chart types
  • 11 supported diagram types
  • Export as image
  • Zooming and panning (In Development)
  • Highly interactive
  • Responsive (In Development)

Grid and Pivot Features (In Development)

  • Conditional formatting
    • Place icons in cells for the given conditions!
    • Colorize cells gradiently with declaring min and max!
    • Set excel-like bars in cells!
  • Customizable Columns
  • Responsive
  • Sparklines
  • Export as CSV, XLS

Getting Started

Including Eve

All you need is to include EVE on the page. You can directly include script file like below.

<script type="javascript">your-script-path/eve.charts.min.js</script>

Creating a chart

To create an EVE chart, you need a container element which will contain the chart. Below you can find an example.

<div id="chart1"></div>

Once the container is defined, we are ready to create a chart in it. You can configure nearly everything of your chart. Here is a simple pie chart as an example.

//create a pie chart
eve('#chart1').pie({
    data: [
        { countryName: 'Turkey', population: 84326108 },
        { countryName: 'Germany', population: 81083600 },
        { countryName: 'United Kingdom', population: 64800000 },
        { countryName: 'Italy', population: 60788845 }
    ],
    series: [
        {
            titleField: 'countryName',
            valueField: 'population'
        }
    ]
});
