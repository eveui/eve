Getting Started

EVE is planned to be a visualization library that includes charts, diagrams and data related algorithms. Eve's charts and diagrams are built with d3.js framework. EVE will save your time by letting you skip learning and coding with D3 and yet it will allow you to customize everything whatever you need.

Including Eve

All you need is to include EVE on the page. You can directly include script file like below.

<script type="javascript">eve-path</script>

If you're using an AMD loader for JavaScript modules, you can require it like below.

require(['eve-path'], function(eve){
    //use eve
});

Chart Options

Eve charts are highly customizable. Below you can find customizable properties of a chart.

Property Type Description Default Value
animationDuration number Duration of chart animations, in milliseconds. 500
backColor string Background color of the chart. White
balloon object Sets balloon properties. Balloon Options
container string Container element of the chart.
data array Represents chart data.
height number Represents chart height. auto
legend object Represents legend properties. Legend Options
series array Represents chart series.
width number Represents chart width. auto
xField string Represents x axis field name for XY charts.
xAxis object Represents X axis properties. Axis Options
yAxis object Represents Y axis properties. Axis Options

Creating an Area Chart

To create an EVE chart, you need a container element which will contain the chart.

<div id="dvAreaChart"></div>

Once the container is defined, we are ready to create a chart in it. 

eve.areaChart({
    container: 'dvAreaChart',
    data: dataXY,
    xField: 'id',
    series: [
        { yField: 'salary' },
        { yField: 'bonus' }
    ],
    yAxis: { title: 'Salary vs Bonus' }
});

Creating a Bar/Column Chart

To create an EVE chart, you need a container element which will contain the chart.

<div id="dvBarChart"></div>

Once the container is defined, we are ready to create a chart in it.

Stacked Bar Chart

eve.barChart({
    container: 'dvBarChartStacked',
    data: dataXY,
    xField: 'state',
    series: [
        { yField: 'salary' },
        { yField: 'bonus' }
    ],
    xAxis: { title: 'States' },
    yAxis: { title: 'Salary vs Bonus' }
});

Grouped Bar Chart

eve.barChart({
    container: 'dvBarChartGrouped',
    data: dataSliced,
    xField: 'state',
    series: [
        { yField: 'salary' },
        { yField: 'bonus' }
    ],
    xAxis: { title: 'States' },
    yAxis: { title: 'Salary vs Bonus' }
});
